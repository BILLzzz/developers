Search engine optimization

If you want to show your Trustpilot reviews on our webpage the widget is a great and easy way of showing them. But the widget is an iFrame solution and therefor you will not be any SEO juice from all the nice reviews. For that reason Trustpilot also provide a JSON feed with the same information as the widget just in JSON format. This feed can be used to build the same widget and just for the propose of getting SEO for the reviews or it can be used for building a widget that support you design the way you thing it will best support your business.
